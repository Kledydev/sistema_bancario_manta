# Sistema de Atención Bancaria - Sucursal Manta

Este sistema simula el flujo de atención a clientes en una sucursal bancaria utilizando estructuras de datos lineales implementadas desde cero en Python 3.

## Estructuras de Datos Implementadas
1. **Cola FIFO (`estructuras/cola_fifo.py`):** Para la cola regular de clientes por orden de llegada.
2. **Cola de Prioridad (`estructuras/cola_prioridad.py`):** Para atender prioritariamente a adultos mayores y personas con discapacidad según normativas ecuatorianas.
3. **Lista Enlazada (`estructuras/lista_enlazada.py`):** Para guardar el historial cronológico de operaciones realizadas en ventanilla.

## Requisitos y Ejecución

- Ejecutar el comando:

python main.py
